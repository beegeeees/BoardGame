# 불편마불 게임 설명서

## 게임 소개
불편마불은 2~4인이 함께 플레이하는 멀티플레이 보드게임이다.

플레이어들은 보드판을 이동하며 점수를 얻고, 라운드마다 진행되는 미니게임에서 경쟁한다.
총 3라운드 종료 후 가장 높은 점수를 가진 플레이어가 승리한다.

---

# 게임 진행 방식
게임은 아래 순서로 진행된다.

1. 플레이어들이 순서대로 턴 진행
2. 모든 플레이어 턴 종료
3. 미니게임 진행
4. 점수 반영
5. 다음 라운드 진행

총 3라운드 진행 후 게임 종료

---

# 플레이어 턴
한 플레이어의 턴은 아래 순서로 진행된다.

1. 카드 사용
2. 주사위 굴리기
3. 말 이동
4. 칸 효과 적용
5. 턴 종료

---

# 주사위 시스템
주사위는 1~6 사이 숫자가 나온다.

플레이어는 원하는 숫자 범위를 어느 정도 조절할 수 있다.

- 1~2 버튼
- 3~4 버튼
- 5~6 버튼
- 랜덤 버튼

일정 확률로 버튼이 섞이거나 가려지는 불편한 UI 기믹이 등장한다.

---

# 점수 규칙
이동한 칸 수만큼 기본 점수를 얻는다.

예시:
- 1칸 이동 → +1점
- 6칸 이동 → +6점

추가로 도착한 칸 효과에 따라 점수가 변한다.

점수는 음수가 될 수 있다.

---

# 보드판 구성
보드판은 총 16칸으로 구성된다.

## 칸 종류
### 시작 칸
- 시작 위치
- 도착 시 +5점

### 점수 칸
- +3점 획득

### 감점 칸
- -3점 감소

### 물음표 칸
랜덤 이벤트 발생

예시:
- 점수 교환
- 위치 교환
- 다음 주사위 강화
- 전체 플레이어 점수 변화

### 카드 칸
카드 1장 획득

### 광고 칸
짧은 미니게임 진행

- 성공 시 +5점
- 실패 시 -5점

---

# 카드 종류
## 방어권 카드
다음 감점 효과를 막는다.

## 10 주사위 카드
주사위 범위가 1~10이 된다.

## 456 주사위 카드
4~6만 나오게 된다.

## 홀짝 주사위 카드
홀수 또는 짝수만 나오게 만든다.

## 광고 스킵 카드
광고 칸을 건너뛴다.

---

# 미니게임
모든 플레이어의 턴이 끝나면 미니게임이 시작된다.

## 미니게임 종류
1. 색상 캡챠
2. 비밀번호 생성
3. 볼륨 미로

## 미니게임 점수
| 순위 | 점수 |
|---|---|
| 1위 | +10 |
| 2위 | +7 |
| 3위 | +5 |
| 4위 | +3 |
| 실패 | 0 |

---

# 승리 조건
3라운드 종료 후 가장 높은 점수를 가진 플레이어가 승리한다.

동점일 경우:
1. 미니게임 1위 횟수
2. 평균 미니게임 순위
3. 마지막 미니게임 순위

그래도 같으면 공동 승리

---

# 게임 특징
- 불편한 UI 기믹
- 빠른 멀티플레이 진행
- 카드와 랜덤 이벤트 중심 플레이
- 미니게임 경쟁 요소
- 점수 기반 승리 시스템
